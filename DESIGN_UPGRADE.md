- ✨ 边框发光效果

## 🛠️ 技术实现

### 使用的CSS特性
- **Glassmorphism**: `backdrop-filter`, `rgba` 透明度
- **渐变**: `linear-gradient`, `radial-gradient`
- **动画**: `@keyframes`, `animation`, `transition`
- **现代布局**: CSS Grid, Flexbox
- **自定义属性**: CSS Variables (`--variable`)
- **伪元素**: `::before`, `::after` 用于装饰
- **响应式**: Media Queries

### 字体
- **Inter**: Google Fonts，权重300-700
- 回退字体: System UI字体栈

### 浏览器兼容
- ✅ Chrome/Edge (最佳体验)
- ✅ Firefox
- ✅ Safari (webkit前缀)
- ⚠️ 旧版浏览器可能不支持backdrop-filter
